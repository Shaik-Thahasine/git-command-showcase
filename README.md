# git-command-showcase


Clone Repository

   git clone <repository url>
   
   Its is used to clone the project from remote to local repository
   
Check Status

   git status
   
   It is used to check current status of the file in repository
   
Stage Files

   git add.
   
   It is used to add all files in the repository
   
Commit Changes

   git commit -m "message"
   
   It is used to save the changes of staging with the commit message
   
Push Changes

   git push 
   
   It is used to upload local commits to remote repository
   
Pull Latest Changes

   git pull
   
   It is used to send the changes of remote repository to local repository
   
Create Branch
    
   git checkout -b branch-name
    
   It used to create a new branch and switch 
    
Git Stash

   git stash
   
   It is used to pass the work and save temporary.
